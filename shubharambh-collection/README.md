# ✦ Shubharambh Collection ✦
### Premium Indian Ethnic Wear — Video Shopping Experience

Welcome to the official repository of **Shubharambh Collection**, a state-of-the-art e-commerce platform designed for a premium, interactive shopping experience. 

![Shubharambh Cover](/public/images/shop/hero-bg.png)

---

## 🚀 Key Features

- **🎬 Video Reels Shopping**: High-quality product reels with audio, allowing customers to see the fabric in motion.
- **📱 Mobile-First Design**: Fully responsive UI optimized for the best smartphone shopping experience (Flipkart/Myntra style).
- **☁️ Supabase Powered**: Real-time inventory management, cloud storage for media, and secure order tracking.
- **💬 WhatsApp Integration**: Instant checkout and order confirmation directly through WhatsApp.
- **🔐 Admin Dashboard**: Comprehensive control panel to manage products, upload reels, and monitor orders.

---

## 🛠️ Tech Stack

- **Frontend**: [Next.js 15+](https://nextjs.org/) (App Router), React 19
- **Database & Auth**: [Supabase](https://supabase.com/)
- **Styling**: Modern Vanilla CSS Design System (Custom Glassmorphism)
- **State Management**: React Hooks & Context API
- **Deployment**: Vercel

## ⚙️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kunal898/kunal-ecommerce-projects.git
   cd shubharambh-collection
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env.local` file in the root directory and add your Supabase credentials:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
   NEXT_PUBLIC_WHATSAPP_NUMBER=919766530018
   ```

4. **Run Project**
   ```bash
   npm run dev
   ```
